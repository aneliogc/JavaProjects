O objetivo desse exercicio é criar uma lista de palavras em ingles que possuem o nome da palavra e a tradução em portugues. 
Essas palavras devem ser definidas via código. Com essas palavras ja criadas, o programa ira exibir aleatoriamente as palavras,
e o usuario devera digitar a tradução da palavra, então o programa verifica se a tradução está correta ou não. O programa não
deve exibir palavras repedidas, e cada palavra tem apenas uma tradução. Quando todas palavras ja tiverem sido exibidas, o programa
deve mostrar quantas palavras o usuario acertou e a porcentagem de acerto.
